Introduction to the website

The website that I have designed is the academic academic-related website. This website is developed using HTML, CSS, and Javascript programming language. My website provides educational information as our website is developed for educational institutes. The website is built for Mantra Academy who are providing education services since 2000 A.D. As they want to introduce themselves worldwide they have hired my team to develop a website for them.



Objective

Before further enhancing your project you need to set certain objectives of your project so that it gives a path to achieve the goal that you have set. So, there are certain objectives for developing this website too, which are listed below:

  • To provide quality information about the organization and courses to everyone unable to gather information about your organization physically

  • To help organizations connect to people through the internet

  • To promote digitalization and attract new students as new generations are more attracted towards digitalization

  • To grow interaction between the organization and students

  • To grow interaction between the organization and students



Problems faced by the organization before website development

Before developing a website we need to identify the problems with existing sites. This helps you to identify what you need to include in your website, which helps you to achieve the objectives. Some of the problems that have been faced by the organization in the absence of the organization’s website are as follows:

  • The institution was not able to provide depth information about their organization to the people.

  • Most of the people were unknown about this organization

  • The organization was not able to get connected to interested students

  • Student who lives far from the organization face difficulty in connecting with the organization



How thie website will solve their problem

Considering all the complications within the organization, lack of marketing and customer interactions were some serious issues. Therefore, the organization hired us to seek solutions to their problems. So, we have developed the website in such a way that it overcomes all the problems.

  • Students can easily find out the information about the organization

  • New families can get admission for their children

  • A student who is far from the organization can connect to get information

  • Organizations can get feedback on the services and facilities that they provide



However, our team faced multiple technical difficulties during the project. Compatibility in browsers was lacking which at first, was solved after following the rules of makeup validation. Likewise, our team also faced internet-related issues as the connection was too weak to check web page loading. The use of too many rich media and images increased the size of webpages which slowed down the speed of webpage loading. This was solved by minimizing the sizes of used files.

Programming Languages Used



HTML(Hypertext Markup Language)

This is one of the most commonly used languages. This was first introduced in 1991 by Tim Berners- Lee ( V.Arya, R.Yankee,2020). This was upgraded from time to time. The last time it was upgraded was in 2014 from XHTML to HTML5(V.Arya, R.Yankee,2020). Various tags and elements are used to display the content properly.



CSS(Cascading Stylesheet)

This language is used to style the HTML content. For example, using this technique, the background of the page, font size, color, text style, and many other styles of content inside html are done. The tools that are used to write CSS code are also similar to that of HTML. It is also written in sublime text or any other text editor where HTML code is written. There are various ways to add CSS to the HTML pages are described below:

  • Inline CSS: This is the type of CSS that can be written inside the HTML code. It is easy to write. It is written without using the selector. But in inline styles, if you want to change the color of all the text inside the h1 tag to blue then you need to style every h1 text which is one of the disadvantages of inline styles. This type of styling of HTML is not used often.

  • External CSS This is a type of CSS that needs a separate CSS file. To style the text in html code we need to introduce a class or id inside the html tags so that using this class and id we can style the text. This style is done externally as we need to open separate files to follow this type of style.

  • Internal or Embedded CSS This type of styling is performed in the head section of the HTML code. So, this type of styling does not need any kind of link or file extension to save the CSS code as it is written inside the HTML code.
