Hourly Weather Analysis

a data analysis project

Sandesh Subedi

February, 2022



Abstract

In this project, data analysis is done, using a dataset named ‘Hourly Weather Data’. The dataset provides weather information about two different airports in the USA : John F. Kennedy (JFK) and LaGuardia Airport (LGA). It consists of 15 columns and about 17,412 rows in total. Based on ‘R Programming’, the provided set of data is explored, manipulated, and analyzed in detail. Necessary data packages are imported along with significant functions, which helped in studying the dataset enormously. The provided set of data is pre-processed in such a way that it provides quality results during analysis. A total of 16 weather analysis is done in the project, including two additional features.
