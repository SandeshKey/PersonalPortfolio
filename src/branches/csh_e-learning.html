E-learning Website with C#

Sandesh Subedi

March, 2021



Abstract

The main aim of this project is to provide a platform through the website which determines the “Web–based Learning System”. The project starts with the identification of requirement specifications for this website. After the identification of requirement specification, entities, attributes, and the relationship between the entities, an ERD (Entity relationship diagram) is formed. Likewise, the required wireframes web pages, and navigational structure for this website are created which leads to the development of consistent pages and a decentralized system. This project is looking forward to serving both non–non-registered as well as registered users who are searching for online learning activities and digital learning resources.
